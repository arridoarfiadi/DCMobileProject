# DC_Mobile 
##Developer: Arrido Arfiadi

iOS Application for UWB Diversity Center
-Acts as personal platform to learn about iOS development as well as a prototype for potential future release

Pods installed (Cocoapods):
* Facebook SDK
* SwipeCellKit
* Chameleon Framework
* Skeleton View
* Realm 
* Firebase
* SVProgessHUD
